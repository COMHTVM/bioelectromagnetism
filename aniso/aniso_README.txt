Code in this directory implements anisotropic diffusion in 2d (images)
and 3d (volumes). Two different algorithms are implemented.

1) Robust anisotropic diffusion following Black et al, IEEE Trans on
Image Processing, 7:421-432, 1998.

2) Affine invariant anisotropic diffusion following Sapiro & Ringach,
IEEE Trans on Image Processing 5:1582-1586, 1996.
